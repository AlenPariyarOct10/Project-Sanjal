## Overview

**ProjectSanjal** is a platform designed to help students showcase their academic projects and discover innovative work from other students.

It provides a centralized place where projects from different colleges and universities can be shared, explored, and discussed.

The goal is to **increase visibility for student work, encourage collaboration, and build a culture of sharing academic knowledge.**

---

## Features

### Project Discovery
- Browse academic projects submitted by students
- Search projects by title or description
- Filter by course, subject, tags, algorithms, and college
- Sort projects by popularity or latest submissions

### Project Showcase
Each project page includes:

- Project description
- Cover image
- Technology tags
- Academic context (course, subject, algorithms)
- GitHub repository link
- Live demo link
- Downloadable documentation and source code
- View and download statistics

### Community Interaction

Users can:

- Like projects
- Comment and discuss projects
- Reply to comments
- Explore related projects

### Student Portal

Students can:

- Create and manage their projects
- Upload documentation and source code
- Create or join teams
- Manage their profile and academic affiliation
- Track engagement on their projects

### Teams & Collaboration

Students can:

- Create teams
- Invite members
- Collaborate on projects
- Showcase team profiles

### Admin Management

Administrators can manage:

- Users
- Projects
- Colleges and universities
- Courses and subjects
- Tags and algorithms
- Platform configuration
